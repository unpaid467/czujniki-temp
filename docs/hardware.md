# Hardware

## Current Hardware

Board:

- LilyGO TTGO LoRa V1.3

Main components on the board:

- ESP32
- SX1276 LoRa 868 MHz
- 0.96 inch OLED

External sensor module:

- BME/BMP280
- pins: VIN, GND, SCL, SDA

## Current Sensor Status

The BME/BMP280 pins are not soldered yet.

Do not connect or test the sensor in software until the pins are soldered and
the wiring is checked.
